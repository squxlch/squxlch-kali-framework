- `newlab`: Create a standardized workspace
- `recon`: Run and save common Nmap scans automatically
- `webscan`: Launch your preferred web enumeration tools and organize the output
- `loot`: Add or index credentials and artifacts from the command line.
- `report`: Generate a report skeleton from your notes.
- `doctor`: Check that required tools are installed and up to date.
